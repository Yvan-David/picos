# Picos Rwanda

A modern, elegant, and responsive website built for an interior design firm, designed by **thetechers**.

This project was built to showcase a full range of interior design services, portfolios, and online booking functionalities using modern web technologies.

## 🚀 Tech Stack

This project is built using a modern React ecosystem:
- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components**: [Radix UI](https://www.radix-ui.com/) (headless accessible components) & shadcn/ui patterns
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Forms**: [React Hook Form](https://react-hook-form.com/) 
- **Carousels**: [Embla Carousel](https://www.embla-carousel.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## ✨ Key Features

- **Responsive Design**: Beautiful layouts across mobile, tablet, and desktop devices.
- **Dynamic Animations**: Smooth scroll reveals, page transitions, and interactive UI elements via Framer Motion.
- **Service & Portfolio Showcases**: Clean, organized display of interior design projects and service offerings.
- **Modern Routing**: Utilizes Next.js App Router for optimal performance and fast page loads.
- **Accessible UI**: Keyboard navigable and screen-reader friendly components powered by Radix UI.

## 🛠️ Getting Started

To run this project locally, follow these steps:

1. **Install dependencies:**
   Make sure you have Node.js installed, then run:
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **View the application:**
   Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## 📦 Build for Production

To create an optimized production build:
```bash
npm run build
```
And to start the production server:
```bash
npm run start
```
